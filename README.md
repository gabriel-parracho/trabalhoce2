# Quickstart

Create a `virtualenv`

```
virtualenv venv
```

Activate the `virtualenv`

```
source venv/bin/activate
```

Install the `requirements.txt`

```
pip install -r requirements.txt
```

Put some netlists `.txt` file in the same directory where `run.py` is

Launch the program

```
python run.py
```
